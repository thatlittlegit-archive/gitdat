Welcome to gitDat

...but just what is it?

GitDat is...

A decenteralized GitHub

Using PouchDB and Dat, GitDat provides a decenteralized version of GitHub.

Open Source

GitDat is fully open source, on both GitHub and GitDat.

Unhosted

There's no 'main' server you must use - the only thing is a central CouchDB, and you can change that.

You can even host your own login server!
